# Simple Interest Calculator

## Project Description
This project is a simple Bash script designed to calculate the simple interest based on user input for principal, rate of interest, and time period.

## How to use:
1. Run the script: `bash simple-interest.sh`
2. Enter the required values when prompted.
3. The script will output the calculated simple interest.

## Formula:
The formula used for the calculation is:
Simple Interest = (Principal * Rate * Time) / 100
